# SecureBank CI/CD Pipeline Project

SecureBank is a production-grade fintech banking microservices application deployed on AWS EKS with a complete CI/CD pipeline using Jenkins.

## Tech Stack
- AWS (EKS, ECR, EC2, RDS, S3, CloudWatch)
- Jenkins CI/CD
- Docker
- Kubernetes
- Helm
- Terraform
- SonarQube
- Trivy
- OWASP ZAP
